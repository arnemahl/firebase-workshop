# Getting Started

1. LOG IN
    1. Go to [console.firebase.google.com](http://console.firebase.google.com)
    1. Log in or create user

2. CREATE NEW PROJECT
    1. Click the button to create new project
        1. Give it a name
        1. (Choose region)
        1. Confirm

3. SET UP CLI
    - `$ npm install -g firebase-tools`
    - `$ firebase login`
